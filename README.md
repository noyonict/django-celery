# Django-Celery
Schedular job with Celery and RabbitMQ

# Requirements
```sh
$ celery==4.4.2
$ Django==3.0.4
```

# Installation Guide Using Dockerfile
1. Follow the commends
```sh
$ git clone https://github.com/noyonict/django-celery.git
$ cd django-celery
$ docker-compose up
```
2. Then browse: 127.0.0.1:8800
4. All done
